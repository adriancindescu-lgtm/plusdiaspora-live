# PlusDiaspora Live

Repository folosit pentru publicarea site-ului PlusDiaspora prin cPanel Git Version Control.

Flux propus:

1. Codex pregateste site-ul si pachetul ZIP final.
2. ZIP-ul este incarcat in Google Drive.
3. cPanel ruleaza deploy-ul din `.cpanel.yml` si publica continutul in `/home/plusdiaspora/public_html`.

Repository: https://github.com/adriancindescu-lgtm/plusdiaspora-live

Site live: https://www.plusdiaspora.com
